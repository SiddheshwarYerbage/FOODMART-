🛒 FoodMart – Online Grocery Store

A modern and responsive E-commerce Grocery Website built using HTML, CSS, and JavaScript. FoodMart allows users to browse products, add items to cart, and experience a simple online shopping interface.

📖 Description
FoodMart is a front-end based online grocery store website designed to simulate a real-world e-commerce platform. Users can explore various food products, view details, and manage items in their shopping cart.
This project demonstrates essential web development concepts such as responsive design, dynamic cart functionality, DOM manipulation, and UI/UX structuring for e-commerce applications.

🚀 Features
🛍️ Browse grocery products
🛒 Add to Cart functionality
➖ Increase / Decrease quantity
💰 Dynamic total price calculation
📱 Fully responsive design
🎨 Clean and modern UI

🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
